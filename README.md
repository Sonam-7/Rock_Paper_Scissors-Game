# Rock_Paper_Scissors-Game
An interactive Rock Paper Scissors game developed using HTML, CSS, and JavaScript. Features include real-time gameplay, score tracking using local Storage, sound effects, and confetti animations for enhanced user experience. This project demonstrates strong fundamentals of DOM manipulation, event handling, and game logic implementation.
